## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript and PrismaIO starter repository.

## Project Description

System CRUD book using NestJs, typescript and PrismaIO.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
